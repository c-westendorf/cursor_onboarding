## Overview

This section maps your team's growth with Cursor from beginner to expert, with clear milestones and learning paths.

## The Journey

```
Level 1          Level 2            Level 3           Level 4
Beginner    →    Intermediate   →   Advanced      →   Expert
(Weeks 1-2)      (Weeks 2-4)        (Month 2+)        (Month 3+)
```

| Level | Focus | Milestone |

## Quick Assessment

### You're at Level 1 if:
- [ ] Still learning basic Cursor features
- [ ] Haven't written custom rules yet
- [ ] Not sure how context works

### You're at Level 2 if:
- [ ] Have working rules for your project
- [ ] Understand rule types (always/auto/demand)
- [ ] Can manage context effectively

### You're at Level 3 if:
- [ ] Have well-organized workspace structure
- [ ] Using MCP integrations
- [ ] Contributing to team standards

### You're at Level 4 if:
- [ ] Building custom MCP servers
- [ ] Contributing to org marketplace
- [ ] Mentoring others
